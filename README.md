# Stipendio Netto Calcolatore

An open-source, private, and offline salary calculator for Italy.

## Features
- **Net Salary Calculation**: Calculate your monthly and annual net salary from your RAL (Retribuzione Annua Lorda).
- **Regional & Municipal Support**: Accurate estimates for all Italian regions and municipal tax rates.
- **Family Deductions**: Support for spouse and children a carico.
- **100% Private**: Works entirely offline with no tracking or data collection.
- **Modern UI**: Built with Jetpack Compose and Material 3.

## FOSS Compliance
This application is fully open-source and contains no proprietary blobs or non-free dependencies. It is designed to meet F-Droid's inclusion criteria.

## Build Instructions
To build the app yourself:
1. Clone the repository.
2. Open in Android Studio or use Gradle:
   ```bash
   ./gradlew assembleDebug
   ```
